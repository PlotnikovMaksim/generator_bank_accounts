Программа генерирующая банковский счёт и копирующая его в буфер обмена
ver 0.2

[Установка на Windows]
1) python 2.7 и запись в переменны среды. СМ. мануал http://программисту.рф/python/ustanovka-python-na-windows-7/
2) pip install -r requirements.txt

[Установка на unix]
1) Установите python 2.7
2) pip install -r requirements.txt

[Найтрока]
Откройте на редактирование файл config.ini
Замените bic, bic, valuta, perviy_raz на свои. Проверок на некорректность данных пока нет, так что вставляйте корректные данные

[Запуск]
Двойной щелчок по main.py в проводнике или
python main.py в консоли

Запускалось под Windows 10
